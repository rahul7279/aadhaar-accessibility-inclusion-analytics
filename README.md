# Aadhaar Accessibility & Inclusion Analytics

An industry-grade, privacy-aware decision support system built for UIDAI Hackathon.

## 🔍 Problem
Identifying under-served populations and stress zones in Aadhaar enrolment,
authentication, and demographic updates is currently reactive and fragmented.

## 🚀 Solution
This project introduces:
- Accessibility & Inclusion Risk Scoring
- Differential Privacy–based analytics
- Geo-spatial district-level risk visualization
- Interactive decision dashboard
